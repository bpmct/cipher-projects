Encript a plaintext into binary using a keyphrase!





How it works

If you give one input into the first box, it will detect whether you have inputted English or binary and give you the translation from that to the other.

If you give two inputs, it will use a binary subsitution cipher to encript your text and return the encrypted output.

Note: it still doesn't work with two different types of inputs, and it only works if you input the singular input into the first box.
